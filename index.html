Render's Dashboard. US Vehicle Market Analysys

This project is an Interactive Exploratory Data Analysis (EDA) Dashboard developed in Python using the Streamlit and Plotly libraries. Its main objective is to facilitate the dynamic exploration of a used vehicle dataset. The dashboard allows users to filter information by model, vehicle type, and price range. It provides key market metrics (average price, mileage, and days listed) and features detailed visualizations such as price and mileage distributions, categorical analyses of price by type and condition, and scatter plots that explore the correlation between price, mileage, and days listed. This project demonstrates skills in data cleaning, NaN handling, advanced visualization with Plotly, and the deployment of interactive Data Science solutions.

Zyfra Gold Extraction ML MODEL

This project focuses on the application of Machine Learning (Regression) models to optimize the extraction efficiency of critical minerals, such as gold. Using real-time operational data from the flotation and purification process, an early warning system (DSS) was developed capable of predicting the Rougher and Final Concentrate Recovery. The objective is to reduce industrial process volatility, allowing operators to proactively adjust parameters (e.g., reagent addition) to minimize the loss of valuable material in the tailings. Model validation is performed using the sMAPE metric, ensuring its robustness and accuracy in heavy engineering environments.
